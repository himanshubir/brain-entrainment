# Brain Entrainment

App to guide user into a meditative (theta) state using binaural beats. If someone is in delta, it will play 7 Hz (theta) difference binaural beats to drive them to theta. Similarly, if someone is in beta, it will play alpha binaural beats to bring them to alpha, and then the 7 Hz (theta) binaural beats to bring them to theta
